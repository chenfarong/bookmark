游戏服务器引擎


开发环境安装

mongoDB数据库3.4或者以上版本

目录结构

bin 执行文件 
res 策划的资源文件
plugin 动态插件

bin/default.json 默认配置文件
注意添加删除插件需要修改配置信息

Windows x86_64 平台

visual studio 2017 c++

Centos7 x86_64 平台

make
gcc
gcc-c++

 

创建自己的插件


基础

main  是提供给引擎使用的几个接口调用
plugin 是注册网络请求，其他插件请求命令处理函数；配置表装载，处理的位置。


事件 event

玩家进入/退出，上线/下线

定时器 onTimer

监听游戏对象的所有定时器

网络请求 opc

客户通过网络发过来的请求，在这里编写处理逻辑并返回数据给客户端。
多个插件都注册了相同的客户端请求命令，多个插件都各自处理，注意不同插件之间的处理顺序。

其他插件请求 xipc
