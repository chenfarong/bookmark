

# 游戏服务器（云）引擎

## 设计目标

- 开发到生产无缝

- 开发过程中客户端/策划对游戏服无感的存在（开发模式下一人一服，数据系统立即更新）
 
- 版本更新不停服，玩家不中断，一直可以玩，一直可以创角

- 支持玩家间强交互，和弱交互两种需求

- 游戏类型支持多样性，卡牌/RPG/棋牌等

- 网络协议支持 TCP

- 开箱即用


## 设计原则


- 工厂分工模式架构服务系统

- 消息总线（unix_socket方式连接）

- 服务器功能实现最小化（云存储、云日志、云监测、网络排行榜）

- 标准化 API
